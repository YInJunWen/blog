# node node-globalvar

> Node 中也有一个全局变量，叫做 global，不同于 JavaScript 中的全局变量 window。当你定义给一个全局变量的时候，他也自动成为全局对象的属性，但是在 Node 中你不可能在最外层定义变量，因为所有 ode 变量都是属于当前模块的，而模块本身不是最外层上下文

## \_\_filename

表示当前正在执行的脚本的文件名，输出的是当前文件所在位置的 `绝对路径` ，

## \_\_dirname

表示当前执行脚本所在的目录
